# Personal Website

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Build

```sh
pnpm build
```

### Deploy (Deploys to gh-pages branch and auto deploys to github pages)

```sh
pnpm deploy
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
